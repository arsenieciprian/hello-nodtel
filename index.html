Ce este un SIP Trunk?

Un SIP Trunk este o conexiune virtuală între sistemul tău de telefonie (IP‑PBX/UC) și rețeaua publică (PSTN/PLMN), bazată pe protocolul SIP. Printr‑un singur link IP poți transporta simultan mai multe apeluri, cu rute de intrare (DID‑uri) și ieșire, fără limitările liniilor TDM clasice.

  • Semnalizare: SIP (UDP/TCP/TLS) cu SDP pentru negocierea codec‑urilor.
  • Media: RTP / SRTP pentru streamurile audio; opțional RTCP pentru monitorizare calitate.
  • Securitate: TLS pentru semnalizare, SRTP pentru media; listă IP, autentificare digest, rate limiting, fraud detection.
  • Integrare: IP‑PBX, contact‑center, SBC, platforme UC.

What is a SIP Trunk?

A SIP trunk is a virtual link between your IP‑PBX/UC platform and the public networks (PSTN/PLMN), using the SIP protocol. A single IP link can carry multiple simultaneous calls, with inbound DIDs and outbound routing, avoiding the limits of legacy TDM lines.

  • Signaling: SIP (UDP/TCP/TLS) with SDP for codec negotiation.
  • Media: RTP / SRTP for audio streams; optional RTCP for quality monitoring.
  • Security: TLS signaling, SRTP media; IP allow‑lists, digest auth, rate limiting, fraud detection.
  • Integration: IP‑PBX, contact center, SBC, UC platforms.

Arhitectură & Protocoale

Rețeaua este împărțită logic în domeniul privat (IP‑PBX/UC al clientului) și domeniul public (interconectarea cu PSTN/PLMN). Interfața dintre cele două este trunk‑ul SIP, protejat prin SBC. SIP gestionează semnalizarea (INVITE, BYE, REGISTER), iar RTP/SRTP transportă media. Politicile de securitate includ TLS, criptare media, limitare de rată, listă IP și detecție de anomalii.

Architecture & Protocols

The network is logically split into a private domain (customer IP‑PBX/UC) and a public domain (interconnect with PSTN/PLMN). The SIP trunk is the interface between the two, protected by SBC. SIP handles signaling (INVITE, BYE, REGISTER) while RTP/SRTP transports media. Security policies include TLS, encrypted media, rate limiting, IP allow‑lists, and anomaly detection.
