Specificații Tehnice / Technical Deep Dive

Pentru ingineri: detaliile care fac diferența. Arhitectura noastră este modernă, sigură și conformă RFC.

Ce este un SIP Trunk? [RO]

Un SIP Trunk este o interfață logică între domeniul privat (IP-PBX/UC) și domeniul public (PSTN/PLMN), bazată pe SIP pentru semnalizare și RTP/SRTP pentru media. Un singur link IP poate transporta simultan zeci sau sute de apeluri, cu DID-uri pentru intrare și rutare controlată pentru ieșire.

  • Semnalizare: SIP (UDP/TCP/TLS), mesaje INVITE/ACK/BYE/CANCEL/REGISTER/OPTIONS/UPDATE/PRACK/REFER.
  • Media: RTP/SRTP, RTCP pentru măsurarea calității; suport pentru re-INVITE (hold, schimbare codec).
  • Identitate: P-Asserted-Identity (PAI), From/To, Privacy; prezentare CLI corectă conform E.164.

What is a SIP Trunk? [EN]

A SIP trunk is the logical interface between the private domain (IP-PBX/UC) and the public domain (PSTN/PLMN), using SIP for signaling and RTP/SRTP for media. A single IP link can carry dozens or hundreds of calls concurrently, with inbound DIDs and controlled outbound routing.

  • Signaling: SIP (UDP/TCP/TLS), methods INVITE/ACK/BYE/CANCEL/REGISTER/OPTIONS/UPDATE/PRACK/REFER.
  • Media: RTP/SRTP, RTCP for quality; supports re-INVITE (hold, codec change).
  • Identity: P-Asserted-Identity (PAI), From/To, Privacy; correct CLI presentation per E.164.

Arhitectură & Protocoale [RO]

  • Securitate: TLS 1.2+, SRTP (AES_CM_128_HMAC_SHA1_80/32), DTLS-SRTP, certificate CA.
  • NAT traversal: Media anchoring (SBC), symmetric RTP, rport, keepalive (OPTIONS).
  • Timere: Session-Expires (RFC4028), re-INVITE/UPDATE, 100rel (PRACK) pentru early media.
  • DTMF: RTP events (RFC2833/4733) recomandat; alternativ SIP INFO.
  • QoS: DSCP EF (46) pentru RTP, CS3/AF31 pentru SIP; jitter buffer adaptiv.
  • Redundanță: DNS SRV/A cu priorități/weight; terminări multiple; failover la pierdere OPTIONS.
  • Interconectare: IP allow-list sau autentificare digest; rate limiting CPS.
  • Fax: T.38 sau G.711 pass-through (ptime 20, fără transcoding).
  • Codec: G.711 A/µ-law, G.729, Opus (opțional); evităm transcodarea.
